## Getting Started
### 1. Clone this repository
```bash
git clone https://github.com/johnphealipto/portfolio.git
```
### 2. Install dependencies
```bash
yarn
```
OR
```bash
yarn install
```

### 2. Run the development server
```bash
yarn dev
```
